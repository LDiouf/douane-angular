NOUVEAU

Cliquez sur <<Nouveau>> pour ajouter un dossier de liquidation
1. Cliquez sur <<nouveau>>
2. Choisissez sur <<BE>> le bon d’engagement dont vous souhaitez faire la liquidation
3. Cliquez sur << caractéristiques de la liquidation>> pour renseigner le montant, l’objet de la liquidation et les références (NINEA) du bénéficiaire. Celui-ci comporte généralement 9 caractères alphanumériques.
4. Cliquez sur << pièces justificatives>> pour insérer les pièces requises (facture définitive, PV, bordereau de livraison …)
5. Allez sur l’onglet <<mode de règlement>> pour mettre les informations bancaires du titulaire du marché. Cliquez sur <<mode de règlement>> et choisissez celui qui se rattache à votre dossier de marché. Cliquez sur <<établissement payeur>> pour sélectionner la banque. Pour une recherche rapide mettez % ensuite le nom de la banque et cliquez sur OK. Renseignez le code banque, le code guichet, le numéro de compte et le RIB.
6. Cliquez sur <<enregistrer>> pour enregistrer la liquidation

APPEL D'OFFRE

Cliquez sur <<ajout/mise à jour>> pour saisir ou modifier un dossier d’appel d’offre.
1. Pour un nouveau dossier laissez la fenêtre par défaut sur <<saisir un nouveau dossier de passation>>. Renseignez les champs <<Gestion>>, <<Type marché>>, <<Mode de passation>>, <<Objet appel d’offre>>, <<Chapitre>> et <<libellé lot>>. Cliquez sur <<Enregistrer>>. Le dossier d’appel d’offre est créé.
2. Pour modifier un dossier déjà créé, cochez la case <<modifier un dossier de passation>>. Mettez à jour les informations que vous désirez changer parmi l’année de gestion, le type de marché, le mode de passation, le chapitre budgétaire et le libellé. Puis Cliquez sur <<Enregistrer>>. Le dossier est modifié.
Cliquez sur <<consultation>> pour voir la liste des dossiers d’appel d’offre.
1. Cliquez sur <<saisir>> pour ajouter un nouveau dossier.
2. Cliquez sur <<Mettre à jour>> pour mettre à jour un dossier déjà créé.
3. Cliquez sur << Editer>> pour voir les informations relatives à un dossier d’appel d’offre et/ou imprimer la fiche.
4. Cliquez sur <<fermer>> pour quitter.

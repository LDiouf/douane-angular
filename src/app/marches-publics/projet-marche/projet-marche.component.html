PROJET DE MARCHE

Cliquez sur <<Nouveau>> pour ajouter un nouveau projet de marché.
1. Choisissez la catégorie de dépense (fonctionnement ou investissement) et le type de marché (standard / marché à commande / marché clientèle / avenant / tranches conditionnelles). Cliquez sur <<saisir>>.
2. Sur l’onglet <<spécification projet de marché>>, renseignez les champs exercice, objet, responsable, fonction pour la rubrique <<détails du projet de marché>>. Mettez le ninea du titulaire du marché. Cliquez sur <<+Lot>> pour aller chercher le dossier d »appel d’offre relatif au projet de marché. Sélectionnez ledit dossier. Cochez la case à droite et cliquez sur <<valider>>. Renseignez le montant du lot.
3. Allez sur l’onglet <<imputation et destination>>. Renseignez la section, le programme, le chapitre, la ligne, l’action et l’activité.
4. Allez sur l’onglet <<mode de règlement>> pour mettre les informations bancaires du titulaire du marché. Cliquez sur <<mode de règlement>> et choisissez celui qui se rattache à votre dossier de marché. Cliquez sur <<établissement payeur>> pour sélectionner la banque. Pour une recherche rapide mettez % ensuite le nom de la banque et cliquez sur OK. Renseignez le code banque, le code guichet, le numéro de compte et le RIB.
5. Cliquez sur <<Enregistrer>> pour enregistrer le projet de marché et Editez la fiche descriptive de marché à joindre dans le dossier à déposer. Cliquez sur <<Dossier à transmettre>> pour transmettre les projets de marché créés à l’ordonnateur délégué (OD).
1. Cliquez sur <<ordonnateur>> pour choisir l’OD de votre section. Cochez les cases à droite des dossiers dont vous souhaitez faire la transmission. Cliquez sur <<Créer le bordereau>> puis Imprimez les bordereaux ‘original’ et ‘copie’.
2. Déposez l’ensemble de votre dossier au bureau de votre ordonnateur délégué. Cliquez sur <<consultation/mise à jour>> pour faire le suivi ou modifier votre dossier de marché
1. Cliquez sur <<saisie>> pour saisir un nouveau projet de marché.
2. Cliquez sur <<mise à jour>> pour mettre à jour un projet non encore transmis à l’OD.
3. Cliquez sur <<Editer>> pour afficher/imprimer la fiche descriptive de projet.
4. Cliquez sur <<Fermer>> pour quitter la fenêtre.

AJOUT/MISE A JOUR

1. pour un nouveau dossier laissez la fenetre par defaut sur saisir un nouveau dossier de passation
. Renseignez les champs Gestion Type marché Mode de passation Objet appel d'offre Chapitre et Libellé lot
.Cliquez sur Enregistrer le dossier d'appel d'offre est créé
2. Pour modifier un dossier déjà créé, cochez la case modifier un dossier de passation
.Mettre à jour les information que vous désirez changer parmi l'année de gestion, le type de marché, le mode de passation, le chapitre budgéétaire et le libellé puis cliquez sur Enregistrer
.Le dossier est modifié
.Cliquez sur consultation pour voir la liste des dossiers d'appel d'offre
2.1-Cliquez sur saisir pour ajouter un nouveau dossier
2.2-Cliquez sur Mettre à jour pour mettre à jour un dossier déjà créé
2.3-Cliquez sur Edition pour voir les informations relatives à un dossier d'appel d'offre et/ou imprimer la fiche
2.4-Cliquez sur fermer pour quiter
